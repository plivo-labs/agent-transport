# Contributing to Agent Transport

Thanks for your interest in contributing to Agent Transport!

## Getting Started

See the [README](README.md) for prerequisites, setup, build, and test instructions.

## Making Changes

1. Create a branch off `main`.
2. Make your changes. Keep PRs focused — one feature or fix per PR.
3. Run `cargo fmt` and `cargo clippy` before pushing.
4. Run relevant tests before pushing.
5. Open a PR against `main`.

In your PR description, mention which area your change affects (Rust core, Python, Node) so maintainers can apply the right labels for release notes.
